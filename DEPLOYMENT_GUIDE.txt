PART 4: UPLOAD FILES TO GITHUB
=====================================================================

METHOD A: Upload via Browser (Easiest - No Git needed)
-------------------------------------------------------

STEP 1: On your empty repo page, click "uploading an existing file"
        (You'll see this text in the middle of the screen)

STEP 2: Download all files from Claude to your computer first.
        You need these files on your local computer:
        - app.py
        - requirements.txt
        - README.md
        - LICENSE

STEP 3: Drag all 4 files into the GitHub upload box

STEP 4: Scroll down to "Commit changes"
        - Title: "Initial commit - HealthBridge AI"
        - Click green "Commit changes" button

STEP 5: Now create the folders. Click "Add file" → "Create new file"

STEP 6: For .streamlit/config.toml:
        - In filename box type: .streamlit/config.toml
        - GitHub will auto-create the folder
        - Copy-paste the content from config.toml file
        - Click "Commit changes"

STEP 7: For tests/test_rag.py:
        - Click "Add file" → "Create new file"
        - In filename box type: tests/test_rag.py
        - Copy-paste the content from test_rag.py
        - Click "Commit changes"

STEP 8: For .github/workflows/ci.yml:
        - Click "Add file" → "Create new file"
        - In filename box type: .github/workflows/ci.yml
        - Copy-paste the content from ci.yml
        - Click "Commit changes"

STEP 3: Click "New app" button (top right)

STEP 4: Fill in the form:
  - Repository:    Basitali2079/healthbridge-ai
  - Branch:        main
  - Main file path: app.py
  - App URL:       healthbridge-ai  (this becomes your URL)

STEP 5: Click "Deploy!" button

STEP 6: Wait 2-3 minutes. Streamlit installs packages automatically.

STEP 7: Your live URL will be:
  https://healthbridge-ai.streamlit.app
  OR
  https://basitali2079-healthbridge-ai-app-XXXXX.streamlit.app

STEP 8: Copy this URL and add it to your portfolio website!

OPTION A: Streamlit Secrets (Recommended for demo)
---------------------------------------------------

1. In Streamlit Cloud dashboard, click your app

3. Add this:

Then update app.py line where api_key is read:

  api_key = st.sidebar.text_input(...) or os.environ.get("ANTHROPIC_API_KEY", "")

PART 9: COMMON PROBLEMS & FIXES
=====================================================================

Problem: App shows "ModuleNotFoundError"
Fix: Make sure requirements.txt is uploaded. Check spelling exactly:
     streamlit>=1.32.0
     requests>=2.31.0
     pandas>=2.0.0

Problem: App shows blank/white page
Fix: Check .streamlit/config.toml is in the right folder

Problem: "Repository not found" in Streamlit
Fix: Make sure repo is set to PUBLIC on GitHub

Problem: CI/CD shows red ✗ on GitHub
Fix: This is okay for now - tests need pytest. The app still works.

Problem: Dark theme not applying
Fix: Make sure .streamlit/config.toml is uploaded in a folder 
     called ".streamlit" (with the dot)
